Connecting to Web Services

Web services you can connect to include:

  • WFS: http://maps.gcc.tas.gov.au:8080/geoserver/GCC_cc/ows?service=wfs&version=1.1.0&request=GetCapabilities
  • WMS: http://maps.gcc.tas.gov.au:8080/geoserver/GCC_cc/gwc/service/wms?version=1.1.1&request=GetCapabilities
Some documentation for connecting to these can be found here:
  • Leaflet: for leaflet, use a TileLayer (WMS). We have examples in our maps section.
  • MapInfo: see this site: MapInfo WMS Setup for WMS. WFS is similar.
  • QGIS: The above URLs work in QGIS. Try this Spatial Thoughts tutorial.
  • ArcGIS: See the documentation for WMS or WFS.
